# vscode_snippet
 This is a code snippet completion plan. The code was transplanted from Mingdeyang.

## Usage
 It's easy to use this. 
 * Just launch VsCode. 
 * Click the "gear" icon. 
 * Then click "User Snippets". 
 * Copy my code and paste in your .json file.
